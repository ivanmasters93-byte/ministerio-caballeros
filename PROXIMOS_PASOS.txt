PASO 1: CREAR 3 CUENTAS GRATIS (20 minutos)
────────────────────────────────────────────

1. Railway (Base de Datos)
   └─ Sitio: https://railway.app
   └─ Qué hacer:
      • Sign up gratis
      • Create → PostgreSQL
      • Obtén DATABASE_URL
      • Copia a: .env.local
   └─ Tiempo: 5 minutos

2. Mailgun (Email)
   └─ Sitio: https://mailgun.com
   └─ Qué hacer:
      • Sign up gratis
      • Obtén API KEY
      • Obtén DOMAIN
      • Copia a: .env.local
   └─ Tiempo: 5 minutos

3. Clerk (Autenticación)
   └─ Sitio: https://clerk.com
   └─ Qué hacer:
      • Sign up gratis
      • Crea app Next.js
      • Obtén SECRET KEY
      • Obtén WEBHOOK SECRET
      • Copia a: .env.local
   └─ Tiempo: 10 minutos

Otros servicios (Ya están listos):
   ✓ PostHog (obtén key, copia a .env.local)
   ✓ Pinecone (obtén keys, copia a .env.local)
   ✓ Vercel (conectarás después)

PASO 2: CONFIGURAR .env.local (5 minutos)
──────────────────────────────────────────

Archivo: .env.local

Busca estos placeholders y reemplázalos:

DATABASE_URL="postgresql://..."
   ↓ Reemplaza con valor de Railway

MAILGUN_API_KEY="key-..."
   ↓ Reemplaza con valor de Mailgun

MAILGUN_DOMAIN="mg...."
   ↓ Reemplaza con valor de Mailgun

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY="pk_..."
   ↓ Reemplaza con valor de Clerk

CLERK_SECRET_KEY="sk_..."
   ↓ Reemplaza con valor de Clerk

CLERK_WEBHOOK_SECRET="whsec_..."
   ↓ Reemplaza con valor de Clerk


PASO 3: MIGRAR BASE DE DATOS (10 minutos)
──────────────────────────────────────────

Ejecuta en terminal:

$ npx prisma migrate deploy

O si es primera vez:

$ npx prisma migrate dev --name init

Cargar datos de prueba (opcional):

$ npx prisma db seed

Ver datos (interfaz gráfica):

$ npx prisma studio


PASO 4: PROBAR LOCALMENTE (20 minutos)
──────────────────────────────────────

En terminal:

$ npm run dev

Luego:
1. Abre browser: http://localhost:3000
2. Prueba login con Clerk
3. Verifica que funciona todo
4. Ctrl+C para detener servidor


PASO 5: DEPLOY A VERCEL (10 minutos)
────────────────────────────────────

Opción A - CLI:
$ npm install -g vercel
$ vercel

Opción B - Dashboard (Recomendado):
1. Visita: https://vercel.com
2. Sign up con GitHub
3. Importa este proyecto
4. Agrega variables de entorno
5. Deploy automático

PROBLEMA? SOLUCIÓN
================================================================================

"No sé dónde obtener DATABASE_URL"
   └─ Railway Dashboard → Settings → Database → Connection String

"Mailgun no me deja crear cuenta"
   └─ Necesitas verificar email + crear dominio

   └─ Debes crear el webhook en Clerk Dashboard (Webhooks section)

   └─ Verifica que .env.local esté completo y correcto

"Build falla"
   └─ npm install
   └─ npx prisma generate
   └─ npm run build

"BD no connecta"
   └─ Verifica DATABASE_URL en .env.local
   └─ Verifica que Railway esté running
